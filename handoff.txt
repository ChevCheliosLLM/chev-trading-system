## HOW DEXTER WORKS
================================================================================

Every 2 minutes Dexter scans all pairs across timeframes:
- Crypto: 5m/15m/30m/1h/4h (full Tier A/B SR model, Binance free)
- Forex/Stocks: 15m/30m/1h/4h (Tier B only — protects Twelve Data API budget)

When 3+ confluences align on a pair/TF, Dexter calls Chev via Open WebUI API:
  POST http://localhost:3000/api/chat/completions
  model: "chev-chelios"
  message: [full market brief + candle data + SR levels + Chev's playbook + journal]

Chev replies in strict format:
  POST: <15-word max summary>
  TRADE: direction=long, entry=X, sl=Y, tp=Z, risk_pct=N, leverage=N,
         position_size_usd=N, tags=sr_4h,fib_1h,rsi, trade_type=day,
         sr_level=X, fib_high=X, fib_low=X, vp_start=YYYY-MM-DD,
         rsi_div_t1=YYYY-MM-DD, rsi_div_t2=YYYY-MM-DD
  REASONING: <full analysis, no word limit>

  OR: SKIP: <one sentence why>

Dexter parses the TRADE: line, validates confluence score, checks SL direction,
logs to Google Sheets, and monitors until TP/SL/expiry.

## HOW DEXTER CALLS CHEV (key function: ask_chev_to_judge)
================================================================================

Located at ~line 3590 in dexter.py. Called when scan_pair_tf() detects 3+ confluences.

The message sent to Chev includes:
1. Chev's playbook (last 500 chars)
2. Recent relevant journal entries (last 3 same asset type + last 2 any type)
3. Full market brief: primary TF candles + OHLCV + RSI + EMA + volume trend
4. Higher TF summary (4H context)
5. Dexter's detected confluence zone + reasons
6. Direction hint (near support = long bias, near resistance = short bias)
7. TOOLS AVAILABLE block — explicit instructions to call these before deciding:
   - get_support_resistance(symbol, tf) — ALWAYS, verify Dexter's zone
   - get_stacked_fibonacci(symbol, tf) — if fib retracement visible
   - get_volume_profile(symbol, tf) — if near volume cluster
   - detect_rsi_divergence(symbol, tf) — if RSI diverging
   - get_atr_stop_suggestion(symbol, tf, entry, direction) — SL sanity check
8. Full trading rules (SL placement, RR expectations, pending orders, etc.)
9. ABSOLUTE RULES (SL must be below entry for long, above for short)
10. Confluence scoring table + threshold

If Chev's reply doesn't start with POST:/SKIP:, Dexter retries once with reminder.
